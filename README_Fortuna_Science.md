# 🔮 Fortuna-Science

**Fortuna-Science** is a bilingual open archive exploring fate systems, cognitive rhythm, and the structures of time across civilizations.

이 프로젝트는 고대의 운명 체계와 현대의 인지과학을 통합해  
“태어난 때와 장소가 왜 중요한가”라는 질문에 구조적으로 접근합니다.
